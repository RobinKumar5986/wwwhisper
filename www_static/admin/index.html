A list of addresses (protocol://domain[:optional port]) that can be used to access the site . Requests from addresses that are not on the list are rejected (this is an important part of the Persona based login process).

If https://example.org is on the list and http://example.org is not, requests to http://example.org will be redirected to https://example.org.

It is safe to add http://localhost[:port] to communicate with wwwhisper from a local machine.
